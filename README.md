# BayonettaRuLinux

### Установка:

```bash
bash -c "bash <(wget -qO- https://raw.githubusercontent.com/AnilAntari/BayonettaRuLinux/main/install.sh)"

```


#### Установка xdelta

```bash
#Arch
sudo pacman -S xdelta3

#Debian/Ubuntu
sudo apt-get install xdelta3
```
